# Overview

> Inherits: Node3D

Shorten to `PCam3D` (text) or `pcam_3D` (code), is used in 2D scenes.

⚠️ A scene must contain a [`pcam_host`](https://github.com/ramokz/phantom-camera/wiki/PhantomCameraHost) for the `pcam_3D` node to work.
